# Maquette Template Images

These generated PNGs are optional edit-mode scaffolds for Maquette image artifacts.

- `brand-kit-template-v1.png`: neutral 1:1 brand board frame for palette, type, surfaces, icon tone, motion/state tone, imagery mood, token samples, and a small UI example.
- `component-closeup-template-v1.png`: neutral 1:1 component close-up frame for one focused component family, variants, states, slots, mobile behavior, and motion cues.

Use them to improve artifact structure and inspectability, not to constrain the creative direction. The filled image must still come from the user brief, approved references, and generated concept. If a template makes the output feel generic, crowded, or less faithful, regenerate without it and record the reason.
